# gtmp

## Overview
create file from template

## Usage
install
```
% go get -u github.com/midorigreen/gtmp
```
build
```
% go build
```
run
```
% ./gtmp -l [language]
```

### File placement
resource file
- resource/xxx/base.toml

template file
- template/xxx/[FILE NAME].xxx

## Template supported lang
- Java Test
